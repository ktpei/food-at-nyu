# food@NYU

A mobile app for NYU students to discover dining options around campus. Browse dining halls and discount places, view menus, check hours, and see community ratings.

## Features

- **Interactive Map** - View all dining locations on a map centered around NYU's Washington Square campus
- **Dining List** - Browse and filter dining halls and discount places
- **Search** - Find dining options by name or keywords
- **Details** - View menus, hours, and crowdedness levels
- **Voting** - Upvote or downvote locations to help other students

## Tech Stack

- React Native with Expo
- Firebase (Firestore)
- React Native Maps
- Expo Router

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- Firebase project with Firestore enabled

### Installation

1. Clone the repository
   ```bash
   git clone <repository-url>
   cd food-at-nyu
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Set up Firebase
   - Create a Firebase project and enable Firestore
   - Add your Firebase configuration to `firebase/config.js`

### Running the App

```bash
npx expo start
```

Then choose to run on iOS, Android, or web.
